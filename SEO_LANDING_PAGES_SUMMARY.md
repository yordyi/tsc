### 前端技术栈
- **框架**: Next.js 14 App Router
- **语言**: TypeScript (类型安全)
- **样式**: Tailwind CSS (响应式设计)
- **动画**: Framer Motion (流畅交互)
- **图标**: Lucide React (统一图标)

### 组件架构
